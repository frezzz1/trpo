Глоссарий

**Бонусный балл** — единица учёта, начисляемая за покупки для обмена на скидки.

**Акция** — мероприятие с правилами начисления или списания баллов.

**Клиент** — участник программы лояльности, накапливающий баллы.

**Менеджер** — пользователь, создающий и управляющий акциями.

**Администратор** — пользователь с правами настройки системы и управления доступом.

**Транзакция** — запись о начислении или списании баллов.

**Баланс** — текущее количество доступных баллов у клиента.

**История** — журнал всех транзакций клиента.

**Срок действия баллов** — период активности начисленных баллов.

**Программа лояльности** — набор правил и условий поощрения клиентов.

**Правило начисления** — условие для начисления баллов (например, % от покупки).

**Сегментация клиентов** — разделение на группы для таргетированных акций.

**Уведомление** — автоматическое сообщение о событиях в системе.

**Роль** — набор прав доступа (клиент, менеджер, администратор).

**Аудит действий** — журнал действий пользователей для безопасности.

**Интеграционный контур** — внешние системы (CRM, ERP, платёжные шлюзы).

**API-ключ** — идентификатор для безопасного доступа к API.

**Дашборд аналитики** — панель с графиками и отчётами по метрикам.

**Коэффициент конверсии** — соотношение стоимости балла к реальной валюте.

**Промокод** — код для скидки или бонусных баллов.

**Условие списания** — правило для списания баллов при заказе.

**Мобильное приложение** — клиентское приложение для iOS/Android.

**Панель администратора** — интерфейс для управления системой.
